# U-Net Cookbook

This page is reserved for practical U-Net recipes that build on the synthetic
demo and tested `UNet2D` implementation.

## Planned Recipes

- Synthetic forward-pass variations.
- Binary and multiclass output-channel examples.
- Shape-checking examples for even and odd spatial sizes.
- Safe experimentation with `features` values and parameter counts.

All examples on this page should use synthetic tensors unless a public, properly
licensed dataset is explicitly configured. Do not add private medical images,
PHI, patient identifiers, DICOM headers, or clinical data.
